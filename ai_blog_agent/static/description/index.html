AI Blog Agent

Automated content discovery & SEO blog writing powered by AI

Key Features

Sniffer Agent

Monitors RSS feeds and REST APIs for trending news matching your domain keywords. Supports provider-native web search (Google Gemini) to skip external sources entirely.

Editorial Workflow

Each discovery becomes a proposal with title, summary, editorial angle, sources, and a relevance score. Editors validate or reject proposals before any content is written.

Writer Agent

One click generates a complete, SEO-optimised blog post with H1/H2/H3 structure, meta title, meta description, meta keywords, tags, and an optional cover image.

Supported AI Providers

Google Gemini  ·  OpenAI (GPT)  ·  Anthropic Claude  ·  DeepSeek  ·  Any OpenAI-compatible API
